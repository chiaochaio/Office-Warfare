# 1131418_杜語喬_公司職級之戰 (Office Warfare)
<br>這是一款策略對戰遊戲。主題結合了現代職場文化，將傳統的暗棋邏輯轉化為職場職級壓制，呈現一場精彩的「辦公室生存戰」。 <br> <br>

## 遊戲特色
- **職場主題化**：所有棋子均以公司職級命名（董事長、總經理...實習生）。
- **隨機洗牌系統**：採用 Fisher-Yates 演算法，確保每局開牌的隨機性與公平性。
- **語音互動**：整合 **TTSMAKER** 語音合成技術，包含開局、翻牌、投降及獲勝等專屬配音。
- **特殊規則設計**：除了大吃小，特別加入「實習生逆襲」機制（等級 1 的實習生可吃掉等級 7 的董事長）。
- **完整遊戲流程**：包含陣營判定、回合切換、勝負判定及投降功能。

## 遊戲規則

1. **開局**：棋盤共 16 格，初始皆為蓋牌狀態。
2. **決定陣營**：首位玩家翻開的第一張棋子顏色（紅方/藍方），即為該玩家之陣營。
3. **回合操作**：
   - **翻棋**：點擊棋子，揭曉職級。
   - **移動/吃子**：點擊已方棋子後，可移動至鄰近（上下左右）的空格或敵方棋子。
4. **職級壓制邏輯**：
   - **大吃小**：7 > 6 > 5 > 4 > 3 > 2 > 1。
   - **逆襲**：實習生 (1) 可吃掉 董事長 (7)。
   - **平手**：同職級相撞時，雙方同歸於盡（同時移除）。
5. **獲勝條件**：吃光對方所有棋子，或對方無法移動、自願投降。<br><br>
